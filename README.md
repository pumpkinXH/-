# 基于SpringBoot+Vue实现高校科研项目与成果管理系统

项目更多细节可联系q：835259802

# 技术栈
|     前端技术      | 说明             |
| :---------------: | ---------------- |
|        Vue        | 前端框架         |
|       Vuex        | 全局状态管理框架 |
|     ElementUI     | 前端UI框架       |
|       Axios       | 前端HTTP框架     |
| vue-element-admin | 项目脚手架       |

|   后端技术   |      说明       |
| :----------: | :-------------: |
|  SpringBoot  |  容器+MVC框架   |
|   MyBatis    |     ORM框架     |
| MyBatis-plus | MyBatis增强工具 |
|    Redis     | 非关系型数据库  |
